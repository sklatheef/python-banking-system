# Python Banking System

## Overview
This project simulates a simple banking system built using Python. 
It demonstrates core banking operations such as account creation, deposits, withdrawals, and balance checking.

## Features
- Create new bank account
- Deposit money
- Withdraw money
- Check account balance
- Basic transaction validation

## Technologies Used
- Python
- SQLite

## Project Structure
main.py – Application entry point  
functions.py – Banking operations  
encryption.py – Password encryption  
decorator.py – Authentication decorator  
exceptions.py – Custom exception handling  

## How to Run
Clone the repository and run:

python main.py
